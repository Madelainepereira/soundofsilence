# *Introduction* <img width = 50px, height = 50px src="https://github.com/nati-Bel/Proyecto-3---Music-App/assets/132446932/f963cbe0-2639-4950-81d2-6f41e4f9a7f5">
Sound Recognition for the Hearing Impaired. SoundofSilence is an application designed to assist individuals with hearing impairments in recognizing sounds in their environment. With just a click of a button, the app uses a database to identify and provide four possible results, taking into account the weights assigned to each recognition.


# *Functionalities* <img width = 50px, height = 50px src="https://github.com/nati-Bel/Proyecto-3---Music-App/assets/132446932/cd1cfd60-fde6-423f-a2b0-638571ee0b00">
- **Sound Recognition**: Easily identify various sounds in the environment.
- **Weighted Results**: Get four possible results based on the weights assigned to each recognition.
- **Historical Data**: Save recognition data to improve result accuracy over time.
- **Backend with Python**: Utilizes Python for the backend.
- **Frontend with React**: Developed with React for the frontend.

# *How users can use it* <img width = 50px, height = 50px src="https://github.com/nati-Bel/Proyecto-3---Music-App/assets/132446932/267ff4c9-035a-4a54-8f0f-f708271c3538">
In order to visualize the frontend part of this project properly you need react and axios installed.<br>
First of all please clone the repository:
 - https://github.com/MiriamVarela/soundofsilence  <br><br>

You can run the project with:
  
        npm start
Run the following command to install all the required dependencies for the project:

        npm install
To start the development server, use the following command:
        
        npm run dev
To display it in the browser, open your local host:
  
        http://localhost:5173

The page will automatically reload if you make changes to the code.<br>
You will see the build errors and lint warnings in the console.<br>
